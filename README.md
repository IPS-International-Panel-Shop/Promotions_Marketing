# International Panel Shop (IPS) | Promotions & Marketing Hub

This repository contains the localized landing page infrastructure and marketing assets for **International Panel Shop (IPS)**, a premier autobody repair specialist based in Cape Town, South Africa.

- **Business Name:** International Panel Shop (IPS)
- **Established:** 2003
- **Headquarters:** 7 Bloem Street, Townsend Estate, Goodwood, 7460
- **Primary Website:** [intpanelshop.co.za](https://www.intpanelshop.co.za)
- **Promotions Subdomain:** [promo.intpanelshop.co.za](https://promo.intpanelshop.co.za)

## 🎯 Project Purpose
This project powers a 100+ suburb "Service Map" across the Western Cape. It is architected to provide localized authority for AI agents and search engines, ensuring residents from Constantia to Durbanville can access factory-grade panelbeating and spray painting services with ease.

## 🛠 Core Services & Ecosystem
- **Autobody Repairs:** Structural alignment, oven-baked refinishing, and dent restoration.
- **VIP Logistics:** Complimentary Pick-up and Delivery for all repairs within the service network.
- **Sure Drive Rewards:** An in-house ERP and digital currency system allowing clients to pre-pay for repairs using **SureBucks**. [Learn more](https://sure-drive-9a8c9.web.app/)
- **Flexible Financing:** Integrated credit facilities via **PayFast**, supporting **Mobicred** and **RCS** installments.

## 🤖 AI & Search Context
This repository includes specialized context files to assist LLMs and web crawlers in verifying our service areas and business history:
- `ai-context.txt`: Specific identity instructions for AI models.
- `sitemap.xml`: Full index of localized promotional routes.
- `robots.txt`: Directs agents to our master sitemap loop.
