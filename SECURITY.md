# Security Policy

We take the security of our customers, the Sure Drive ecosystem, and the IPS promotion infrastructure very seriously.

## Supported Versions

We actively maintain and provide security updates for the following versions:

| Version     | Supported          |
|-------------|--------------------|
| 5.1.x       | :white_check_mark: |
| 5.0.x       | :x:                |
| 4.0.x       | :white_check_mark: |
| < 4.0       | :x:                |

---

## Reporting a Vulnerability

If you discover a security vulnerability in any of our systems (including the website, Sure Drive platform, or payment integrations), please report it responsibly.

**Please do not report security vulnerabilities via public GitHub issues.**

### How to Report

Send a detailed report to: **info@intpanelshop.co.za**

### What to Include in Your Report

- **Description**: Clear summary of the vulnerability
- **Steps to Reproduce**: Detailed instructions or a Proof of Concept (PoC)
- **Impact**: Potential consequences (e.g., data exposure, service disruption, etc.)
- **Affected Version(s)**: Which version(s) are impacted

---

## What to Expect

- **Acknowledgment**: You will receive a response within **48 business hours**.
- **Status Updates**: We will keep you informed every **7 days** during investigation and remediation.
- **Resolution**: Once the issue is fixed and deployed, we will notify you.

---

## Important Note

International Panel Shop does **not** operate a public bug bounty program at this time. We do not offer financial rewards for vulnerability reports. This policy is focused on **coordinated, responsible disclosure** to protect our customers and infrastructure.

---

Thank you for helping keep International Panel Shop and the Sure Drive ecosystem secure.
